# C Programming Language


### Preprocessing

  Command for just this stage of the compilation process:
   ```
   gcc -E input_filename -o output_filename
   ```

### Compiling

  Command for just this stage of the compilation process:
   ```
   gcc -c input_filename
   ```

### Assembling

  Command for just this stage of the compilation process:
   ```
   gcc -S input_filename
   ```

### C code

  Sample C code:

	return (0);
